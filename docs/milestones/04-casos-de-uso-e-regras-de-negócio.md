## Casos de uso

![diagrama de casos de uso](../../images/diagram-use-case.svg)

O projeto em questão é constituído pelos seguintes casos de uso:

1. consultar usuário
2. atualizar usuário
3. atualizar report
4. consultar report
5. realizar report
6. consultar ficha
7. atualizar ficha
8. lançar ficha
9. inserir localidade
