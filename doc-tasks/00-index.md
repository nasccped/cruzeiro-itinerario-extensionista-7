# Doc Tasks

O diretório de `doc-tasks` tem como objetivo armazenar o progresso
obtido quando é preenchido algum documento, formulário ou qualquer
outra coisa necessária para a atividade no geral.

Os arquivos aqui dispostos não tem caráter de entrega, apenas
servem como [`milestones`](https://pt.wikipedia.org/wiki/Milestone)
dos objetivos de desenvolvimento e de entrega acadêmica.

Você pode acompanhar a inserção e atualização por meio das datas dos
[commits](https://docs.github.com/pt/pull-requests/committing-changes-to-your-project/creating-and-editing-commits/about-commits)
e/ou pelas datas que ficam posicionadas no canto superior direito
anterior a cada seção de conteúdo (parágrafo, título + parágrafo).
